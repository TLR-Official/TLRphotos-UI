# TLR Photos - 上传页面大改版 PRD

## Overview
- **Summary**: 对上传页面进行全面重构，将上传流程划分为航空、铁路、汽车三大类别，取消强制填写描述和自定义标签，采用预设标签选择系统，并为不同类别提供差异化的上传模板。
- **Purpose**: 提升用户上传体验，规范标签体系，确保数据质量，为后续内容检索和推荐提供标准化元数据基础。
- **Target Users**: 所有登录用户，特别是航空、铁路、汽车摄影爱好者。

## Goals
- 将上传页面顶部划分为航空类、铁路类、汽车类三大入口
- 取消强制填写描述，改为可选
- 取消自定义标签输入，采用预设标签选择系统
- 为不同类别提供差异化的上传模板（通用字段 + 类别专属字段）
- 实现三级标签体系架构（一级分类、二级对象、三级属性）
- 添加安全合规声明和关键词过滤

## Non-Goals (Out of Scope)
- 标签系统的后台管理界面（后续迭代）
- AI辅助标签补全功能（后续迭代）
- 地理围栏预警功能（后续迭代）
- 同义词映射表（后续迭代）

## Background & Context
当前上传页面存在以下问题：
1. 描述字段强制填写，增加用户负担
2. 标签为自由文本输入，导致标签格式不统一
3. 没有类别区分，无法为不同类型的照片提供差异化表单
4. 缺少安全合规声明机制

## Functional Requirements
- **FR-1**: 在上传页面顶部添加航空类、铁路类、汽车类三个分类标签，用户必须选择其一
- **FR-2**: 将描述字段改为可选填写（去掉红色星号和必填验证）
- **FR-3**: 取消自定义标签输入，改为从预设标签列表中选择
- **FR-4**: 根据用户选择的分类，显示不同的上传模板
- **FR-5**: 通用字段（图片信息、拍摄参数）对所有分类可见
- **FR-6**: 类别专属字段（航空机型、铁路型号、汽车型号）仅在对应分类下显示
- **FR-7**: 添加安全合规声明勾选框（强制勾选）
- **FR-8**: 实现关键词黑名单过滤（军事、国防等敏感词汇）

## Non-Functional Requirements
- **NFR-1**: 标签数据存储于独立数据库 `/opt/tlr-photos-ui/data/tags.db`
- **NFR-2**: 前端展示中文标签，后端存储英文Key
- **NFR-3**: 标签系统支持三级架构（一级分类、二级对象、三级属性）
- **NFR-4**: 上传页面响应式设计，适配移动端和桌面端

## Constraints
- **Technical**: React + TypeScript + Tailwind CSS，必须与现有代码风格一致
- **Business**: 标签体系必须严格遵循用户提供的《TLR Photos 民用交通影像标签体系规范文档》
- **Dependencies**: 需要创建独立的标签数据库和相关API

## Assumptions
- 用户已了解不同类别的区别，能够正确选择上传分类
- 标签体系的中英文映射已明确
- 后端能够处理分类字段和结构化标签数据

## Acceptance Criteria

### AC-1: 分类选择功能
- **Given**: 用户进入上传页面
- **When**: 用户选择航空类/铁路类/汽车类标签
- **Then**: 页面显示对应类别的上传模板，且必须选择分类后才能进入下一步
- **Verification**: `human-judgment`

### AC-2: 描述字段改为可选
- **Given**: 用户在填写信息步骤
- **When**: 用户只填写标题，不填写描述
- **Then**: 提交按钮可用，上传成功
- **Verification**: `programmatic`

### AC-3: 预设标签选择
- **Given**: 用户在填写信息步骤
- **When**: 用户从预设标签列表中选择多个标签
- **Then**: 标签正确显示为已选择状态，提交后正确存储
- **Verification**: `human-judgment`

### AC-4: 差异化上传模板
- **Given**: 用户选择航空类
- **When**: 用户查看表单字段
- **Then**: 显示航空类专属字段（机型、机场等），隐藏铁路和汽车专属字段
- **Verification**: `human-judgment`

### AC-5: 安全合规声明
- **Given**: 用户在填写信息步骤
- **When**: 用户未勾选安全合规声明
- **Then**: 提交按钮不可用，提示用户必须勾选
- **Verification**: `programmatic`
